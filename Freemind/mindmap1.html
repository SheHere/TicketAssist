    • Account

      • Password

          - Current Faculty/Staff/Student?

          - Alumni or Emeritus?

          - New User? (ACTC, incoming Freshman, new employee, etc.)

        • Partially Working

          • For when the user is able to log in successfully in one area but not another:

              - Determine where the password works and where it doesn't.

              - Ensure that the issue is not being caused by any sort of "remember my password" features. (TODO: Link to guide for Keychain Assess and Windows equivalent)

              - Ensure internet connectivity.

              - Attempt a password reset. (TODO: Link guide to password reset)

        • Current User

          • For more detailed instructions, see (TODO: Link guide to password resets)

              - Find user in Forefront Identity Manager. (TODO: Link to guide for FIM)  

              - Verify client identity by asking for their birth date and the last four digits of their Social Security Number.  

              - Generate a random 8 character long password on https://www.random.org/passwords/    - Use ADAC to change their password. (TODO: Link guide for ADAC)  

              - Ask user if they would like you to teach them how to change their password on their own. If so, see "How to Change Password" section.  

              - Use the PWD quick ticket that best applies. (TODO: Link guide to WHD, specifically section on quick tickets)

        • Alumni / Emeritus

          • For when the user is either an Alumnus or a Presidential Emeritus:

              - After verifying their identity, perform a password reset (TODO)

              - Assist them in setting up their challenge questions and changing their password on their own (TODO: Link guide to alumni password resets)

        • New User

          • For when the user has not yet assessed their account ever:

              - Ensure the user has claimed their account at http://www.stthomas.edu/new

              - To do so, they will need their St. Thomas ID number. If they do not know this,

            you can tell them it after verifying their identity.

              - If it is already claimed, perform a password reset. (TODO: Link guide to password resets)
